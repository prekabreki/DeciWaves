# DeciWaves

Extract speaker- and subtitle-tagged voice audio from your own PC installs of
Death Stranding (Director's Cut), Horizon Zero Dawn Remastered, and Horizon
Forbidden West — and assemble it into story-ordered MP3 reels.

**Windows-only. Bring your own legally owned game. Ships code, never game content.**

Not affiliated with Guerrilla Games, Kojima Productions, or Sony Interactive
Entertainment. Full README coming with v0.1.0.
